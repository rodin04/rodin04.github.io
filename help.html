Help / Support

User Input with example

First, enter the number of runs you have done, then click Submit.

The tool will then generate empty run rows according to the number you entered. One row is one run!

The input for distance must be in meters, and the input for time must always be in minutes.

For example: 100 meters in 0.125 (not 12.5), or 5000 meters in 20.30 minutes.

The number after the dot always means seconds. For example: 0.30 means 30 seconds.

You can use either a comma ( , ) or a dot ( . ) for decimal numbers – both work fine.

Example

100m in 12,5 sek

5000m in 20,30min (20min and 30sek)

When you have entered all your runs, click Calculate below!

Plot of your runs explanation

The tool has now generated the results of your runs.

It shows the distance, time, your speed in m/s and km/h, your time split every 100 meters, and your pace.

The pace is the time it takes you to run one kilometer.

Important

The last row shows the overall averages of all runs in m/s, km/h, pace, and time per 100 meters.

For example, if the first row speed is 8.00 m/s and the second row is 4.07 m/s, the average is 6.03 m/s.

The tool does the same calculation for km/h, pace, and time per 100 meters.

It also shows the total distance, meaning the full amount of meters you’ve run in total.

(You can hide the units by simply clicking the toggle button.)

Plot of runs with same distances

If you enter the same distance multiple times for your runs, it calculates the average time of all those runs — but only if all the distances are exactly the same.
