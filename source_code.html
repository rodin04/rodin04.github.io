Source & Development

This website was built purely with HTML, CSS, and JavaScript — without using WordPress or any other CMS.
The full source code will be available on GitHub soon.
A link to the repository will be added here shortly.

Development Goals:
• Create a fast and responsive user experience
• Keep the code clean and easy to understand
• Provide useful tools for runners and fitness enthusiasts

Technologies Used:
• HTML5 for the website structure
• CSS3 for styling and responsive design
• JavaScript for dynamic features and interactivity

Future Plans:
• Open-source the project to invite contributions from developers
• Add new features based on user feedback
• Improve accessibility and performance continuously

If you're interested in collaborating or have suggestions, feel free to reach out via the contact page
